# Stratos Claude

Pre-configured Claude Code workspace for Stratos Edge work — Linkup, Apify, and Netrows tools,
ready to go.

## Setup

A Stratos admin sends you a `stratos-startup.md` file via 1Password. Then:

1. Download it (leaving it in your Downloads is fine).
2. Open Claude Code in a new, empty folder.
3. Say:
   > I downloaded the startup package to my Downloads — open it and do what it says.
4. Restart Claude Code in that folder when it tells you to.

That's it — Linkup, Apify, and Netrows are ready to use.
