# Stratos Edge — Claude workspace

## Open it
On this page: **Code** → **Codespaces** → **Create codespace on main**. Wait about a minute. Claude opens in the side panel.

## Sign in (first time only)
1. **Terminal → New Terminal**.
2. Run `claude`, click **Authorize**, then paste the **code** back into the terminal.
3. If the panel still shows signed out: `Cmd/Ctrl + Shift + P` → **Reload Window**.

## Use it
Talk to Claude in the side panel.

## Get the latest tools
`Cmd/Ctrl + Shift + P` → **Rebuild Container**.

Questions: ask Peat.
