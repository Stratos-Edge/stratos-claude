# Example Skill

This is a placeholder so the `stratos-edge` plugin ships with at least one working
capability and can be verified end-to-end after install.

To create a real skill, copy this folder, rename it, and edit the frontmatter:

- `name` — the skill's identifier (kebab-case).
- `description` — when Claude should reach for this skill. Be specific about the trigger.

Then write, below the frontmatter, the instructions Claude should follow when the skill
runs. See https://code.claude.com/docs for the full SKILL.md format.
